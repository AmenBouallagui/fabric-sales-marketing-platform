# Data Generation

This folder will contain the synthetic data generator for the sales and marketing source domains.

## Planned Outputs

- Accounts
- Contacts
- Leads
- Opportunities
- Campaigns
- Campaign interactions
- Products
- Revenue transactions
- Sales activities

## Principles

- Generate realistic but fully synthetic data.
- Avoid secrets, credentials, and real customer information.
- Keep generated data files out of the repository by default.
- Make outputs deterministic when a seed is provided.

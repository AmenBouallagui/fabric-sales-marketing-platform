# 01 Bronze Ingestion Notebook Plan

## Objective

Design a future Microsoft Fabric notebook that reads synthetic source CSV files from Lakehouse Files, validates their structure, adds ingestion metadata, and writes append-friendly Bronze Delta tables.

This Markdown file is a notebook implementation plan. It is not an executable `.ipynb` artifact yet.

## Parameters

Planned notebook parameters:

- `source_base_path`: default `Files/source`.
- `load_date`: load partition date in `YYYY-MM-DD` format.
- `ingestion_run_id`: unique run identifier passed by a pipeline or generated by the notebook.
- `ingested_at`: run timestamp.
- `fail_on_missing_file`: boolean flag, default `true`.

## Read CSV Files

Planned path convention:

```text
Files/source/{source_system}/{entity_name}/load_date={load_date}/{file_name}
```

Example read pseudocode:

```python
def read_source_csv(config: dict, load_date: str):
    source_file_path = (
        f"{source_base_path}/{config['source_system']}/{config['entity_name']}"
        f"/load_date={load_date}/{config['file_name']}"
    )

    return (
        spark.read
        .option("header", True)
        .option("inferSchema", True)
        .csv(source_file_path)
    ), source_file_path
```

## Add Ingestion Metadata

Each DataFrame should be enriched with the standard Bronze metadata columns.

```python
def add_ingestion_metadata(df, config: dict, source_file_path: str):
    source_columns = [c for c in df.columns]
    row_hash_expr = F.sha2(
        F.concat_ws("||", *[F.coalesce(F.col(c).cast("string"), F.lit("")) for c in source_columns]),
        256,
    )

    return (
        df
        .withColumn("ingestion_run_id", F.lit(ingestion_run_id))
        .withColumn("source_file_name", F.lit(config["file_name"]))
        .withColumn("source_file_path", F.lit(source_file_path))
        .withColumn("source_system", F.lit(config["source_system"]))
        .withColumn("entity_name", F.lit(config["entity_name"]))
        .withColumn("load_date", F.to_date(F.lit(load_date)))
        .withColumn("ingested_at", F.to_timestamp(F.lit(ingested_at)))
        .withColumn("source_updated_at", F.to_timestamp(F.col("updated_at")))
        .withColumn("row_hash", row_hash_expr)
    )
```

## Validate Required Columns

Structural validation should happen before writing to Bronze.

```python
def validate_required_columns(df, config: dict):
    missing_columns = sorted(set(config["required_columns"]) - set(df.columns))
    if missing_columns:
        raise ValueError(
            f"{config['entity_name']} is missing required columns: {missing_columns}"
        )

    null_key_count = df.filter(F.col(config["primary_key"]).isNull()).count()
    if null_key_count > 0:
        raise ValueError(
            f"{config['entity_name']} contains {null_key_count} null primary keys"
        )
```

## Write Bronze Delta Tables

Bronze writes should append by default.

```python
def write_bronze_table(df, table_name: str):
    (
        df.write
        .format("delta")
        .mode("append")
        .saveAsTable(table_name)
    )
```

## Log Row Counts

The notebook should log entity-level row counts to support reconciliation and pipeline monitoring.

```python
load_results = []

for config in SOURCES:
    raw_df, source_file_path = read_source_csv(config, load_date)
    validate_required_columns(raw_df, config)
    bronze_df = add_ingestion_metadata(raw_df, config, source_file_path)
    write_bronze_table(bronze_df, config["bronze_table"])

    load_results.append({
        "entity_name": config["entity_name"],
        "bronze_table": config["bronze_table"],
        "source_file_path": source_file_path,
        "row_count": bronze_df.count(),
        "ingestion_run_id": ingestion_run_id,
    })
```
